
  # Titeenit Webpage

  This repository contains the code for the Titeenit Webpage.

  The initial version of the code was generated using Figma Make. The generated code has since been edited and adjusted.

  Original design:  
  https://www.figma.com/design/ZkCrLnTy1J0wWyyJGVwq9H/Titeenit-Webpage

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
